# Videos Folder

Бұл папкаға арнайы видео файлын қойыңыз.

## Қолдану:

1. Видео файлын осы папкаға көшіріңіз (MP4 форматы)
2. `src/components/VideoBlock.jsx` файлын ашыңыз
3. Видеоны import етіңіз:
   ```jsx
   import birthdayVideo from '../assets/videos/birthday.mp4';
   ```
4. `hasVideo` мәнін `true` етіңіз
5. Видео тегіндегі комментарды алып тастап, `src={videoSrc}` қосыңыз
